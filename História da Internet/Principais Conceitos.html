Principais Conceitos

WEB

O termo web faz referência à rede que conecta todos os computadores por todo mundo, a World Wide Web (WWW). Significa um sistema de informações ligadas através de hipermídia (hiperligações em forma de texto, vídeo, som e outras animações digitais), que permitem ao usuário acessar uma infinidade de conteúdos por meio da internet. Para tal, é necessário ligação à internet e um navegador (browser), onde são visualizados os conteúdos disponíveis.

W3C

W3C significa World Wide Web Consortium, é um órgão que tem a finalidade de regulamentar, criar e manter os padrões da WEB. Recebe o apoio de várias empresas, tais como a Microsoft, Samsung, Apple, Google, Mozilla, etc. Elas ajudam a manter suas pesquisas, até mesmo incluindo membros de suas equipes nos grupos de trabalho de cada padrão.

HTML

HTML significa Hypertext Markup Language ou, em português, Linguagem de Marcação de Hipertexto. É uma linguagem base da internet, a qual é amplamente utilizada na criação de website; ela que define como os elementos de uma página devem ser exibidos.

Navegadores

Os navegadores são softwares responsáveis por compilar as páginas de linguagem HTML, permitindo uma visualização interativa pelos usuários, tornando fácil o acesso à Internet. Também são conhecidos pelos termos inglês web browser ou simplesmente browser .
